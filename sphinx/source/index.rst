Indexor
=====================

Indexor is a lightweight Python search engine for local `.txt` and `.md` files.
The project provides recursive indexing with positional postings,
query parsing with quoted phrase support,
required-clause handling through `+` adjacency semantics,
BM25-based ranking for relevant results,
and an interactive CLI interface with Rich-styled output.
It is designed to stay simple, fast, and extensible while remaining
easy to understand and maintain.
